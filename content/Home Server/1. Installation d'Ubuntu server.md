# Installation d’Ubuntu Server en Dual Boot

## 1. Préparation du disque pour le dual boot

Avant d’installer Ubuntu Server, préparer le disque sous Windows pour faire un dual boot :

- Réduire le volume principal (C:) via le **Gestionnaire de disques** de Windows.
- Créer une nouvelle partition de la taille souhaitée (par exemple 350 Go) et la laisser en **non alloué**.
- Pour que l’installateur Ubuntu la voie, formater cette partition en **NTFS** (même si Ubuntu va la reformater ensuite).

## 2. Création de la clé USB d’installation

Depuis un PC Ubuntu Desktop :

- Télécharger l’[ISO d’Ubuntu Server](https://ubuntu.com/download/server).
- Ouvrir l’application **Disques** (GNOME Disks).
- Sélectionner la clé USB, puis utiliser l’option **“Restaurer l’image disque…”** pour y écrire l’ISO.
- Attendre la fin de la copie, puis retirer la clé.

## 3. Démarrage sur la clé USB

- Brancher la clé USB sur le mini PC.
- Accéder au BIOS/UEFI (touche F2, F10, DEL selon la marque).
- Mettre la clé USB en premier dans l’ordre de boot.
- Démarrer sur la clé pour lancer l’installation d’Ubuntu Server.

## 4. Installation d’Ubuntu Server

- Suivre les instructions de l’installateur.
- Connecter le mini PC à Internet (Wi-Fi ou Ethernet).
- Lors du choix du disque, sélectionner la partition NTFS créée précédemment.
- Choisir de la formater en **ext4** et de la monter sur la racine `/`.
- Laisser les autres partitions Windows intactes.

## 5. Installation d’OpenSSH et import des clés SSH GitHub

- Accepter l’installation d’**OpenSSH Server** lorsque l’installateur le propose.
- Accepter l’**import des clés SSH depuis GitHub** :
    - Entrer l’identifiant GitHub.
    - **L’installateur récupère les clés publiques** associées au compte GitHub.
    - Cela permet de se connecter en SSH sans mot de passe, dès la première connexion.

## 6. Finalisation de l’installation

- Ne rien ajouter d’autre lors de l’installation (pas de paquets supplémentaires).
- Terminer l’installation, retirer la clé USB, puis redémarrer le mini PC.
- Dans le BIOS, mettre **Ubuntu** en premier dans l’ordre de boot.
- Le système démarre sur Ubuntu Server.

## 7. Première connexion SSH

- Depuis un PC (sur le même réseau local), se connecter en SSH au mini PC grâce à la clé publique importée depuis GitHub :
    ```bash
    ssh mon_utilisateur@adresse_ip_du_mini_pc
    ```
- La connexion fonctionne sans mot de passe, car la clé publique est déjà installée sur le serveur.
